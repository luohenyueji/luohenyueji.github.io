                                    # [讲座论坛] 碧根果产业现状及产业化开发关键技术

                                    # 1. 全球碧根果产业动态

                                    核桃是我国人工经济林种植面积最大的树种，种植范围有1，1.2亿亩。核桃和山核桃都属于胡桃属，是两个不同的种。山核桃又包括一些品种，其中碧根果源自美国，被称为美国山核桃又因为其壳薄被称为薄壳山核桃。

                                    一方面，碧根果在全球产业来看主要分布在澳大利亚，美国、墨西哥以及南非。这4个区域占全球总产量的99%，其中美国占52%以上。美国、墨西哥、南非、澳大利亚为碧根果出口国。而我国每年进口碧根果初步估计4-5万吨。

                                    另一方面，进口碧根果不仅关税高，运输成本高，且运输时间长。碧根果的含油量高，每100g碧根果的脂肪含量高达74.27克,属于坚果中富可流油的典型代表。在运输的过程中，一旦遇到高温，容易被氧化，不仅影响风味也破坏了营养。

                                    优质碧根果市场价值极高，市场价为40-50元一斤，因此，发展我国碧根果产业化能够解决我国市场需求，还能够帮助农民脱贫致富。

                                    # 2. 美国碧根果产业现状

                                    美国碧根果种植面积约为300亩，但产量极高，在13-15万吨。 100多年来美国农业部育种中心已开展142个杂交组合，释放杂交品种29个。 产业发展的主要技术包括：1品种。2 技术

                                    1. 需要持续不断提供良种的种质资源。
                                    2. 合理控制种间距离，一亩地平均不能超过10株。
                                    3. 树形控制，机械化修剪，修剪顶枝，发展侧枝。
                                    4. 苗圃管理，水肥管理是关键。

                                    # 3. 碧根果产业发展关键技术

                                    # 3.1 立地条件（灌溉与土壤）

                                    碧根果对水分要求极高，在缺水的条件下，即使能丰产也很难稳产。土壤要求较高，但土壤可以通过施肥来改善。

                                    # 3.2 品种选择（因地制宜）

                                    1. 授粉（3-5个品种） 碧根果的雄花和雌花不同时开，且花期短。对授粉品种要求严格。建议二雄、二雌树种混合种植，观察品种的物候期尤其是要求雌、雄树种开花时期要接近。
                                    2. 气候因子 不同的气候区要考虑到品种的选择
                                    3. 市场价格 根据市场需求，考虑种植品种的大小，出仁率，果仁完整度，果仁色泽
                                    4. 早熟性 成熟期从10月初到11月底，要考虑到果实的成熟时期，保证市场供应的稳定性。
                                    5. 产量
                                    6. 大小年指数

                                    # 3.3 容器苗

                                    容器壮苗。

                                    # 3.4 种植密度

                                    不同的种植密度，修剪方式不同。建议种植密度为10米行距8米间距。

                                    # 3.5 树形控制

                                    矮化？ 矮化品种，矮化砧木，矮化技术？都不适用于碧根果。 注意适当修剪。老枝不结果，新枝结果，因此是适时修剪枝条。

                                    # 3.6 果实采后及品质提升

                                    品质决定价值。 目前大多树苗木园的种植品种不一。果实品种不一样，加工程序不一样。加工厂收购要求品种统一，规格统一。

                                    # 4. 碧根果发展机遇与挑战

                                    1. 成功吸收美国经验
                                    2. 良种规模化

                                    # 5. 已开展的主要研究工作

                                    1. 种质资源收集和新种质创建
                                    2. 对引进品质进行筛选和区域性实验，选出适宜的品质在省内区推广
                                    3. 采穗园建立 防止品种混乱。
                                    4. 控根育苗关键技术
                                    • 注重发展须根，根系发达才能吸收更多养分。
                                    • 种子温床催芽，芽苗截根移栽。 种子先沉积处理后，再高温催芽，能提高一个月发芽，保证当年能嫁接，然后切掉主根，移到大盆种。
                                    • 嫁接技术优化，对砧木进行培育
                                    • 生长季芽接技术
                                    • 当年夏季采用方块芽接培育嫁接苗。（ 7，8，9月嫁接）芽接成活率为85%以上，枝接在每年4月，对技术要求高，且效果还没有芽接的好。枝接和高接换头是品种更新的主要方式。
                                    1. 扦插繁殖 硬枝扦插成活率达60%，嫩枝扦插成活率80%以上。
                                    2. 组织培养技术 难度大，但尤为重要。
                                    3. 容器育苗标准化生产技术
                                      • 裸根苗，直接移栽
                                      • 用小容器育苗，容易窝根，比裸根号，但是要切根
                                      • 大规模的容器育苗
                                      • 根据市场需求育苗，培育1-5年生苗

                                    # 6.个人总结

                                    1. 目前我国还没有较好的薄壳山核桃品种，主要靠引进优良品种。
                                    2. 发现产业中存在的问题，学习国外先进技术来解决我国的现实问题。
                                    3. 紧跟国家政策要求，解决实际问题。
                                    4. 美国薄壳山核桃的产业自动化程度非常高，是如何进一步发展信息化的？
